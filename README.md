# Todolist project with React

Link : https://glittering-bonbon-621ab2.netlify.app/

เป็นเว็บไซต์ Todolist ที่ใช้ react ในการทำ สามารถพิมพ์งานที่อยากจะทำลงไป เพื่อเตือนสติตัวเองในอนาคตว่าไม่ให้ลืมทำสิ่งนั้น
มีการเก็บข้อมูล Localstorage เพื่อป้องกันข้อมูลสูญหายเมื่อกดรีเฟรช
# Feature

 - Add Task = เพิ่มงาน
 - Delete Task = ลบงาน

# Tech Stack
 - Frontend : React
 - Style : CSS
